## 详细创建步骤

### 1. 登录并进入工单系统
1. 访问 [Vultr官网](https://bit.ly/VuLtr)
2. 登录您的账户
3. 点击左侧导航栏的"Support"
4. 选择右上角"Open Ticket"按钮

### 2. 填写工单内容
工单需用英文填写，主要包含以下字段：

| 字段名称 | 填写说明 | 注意事项 |
|---------|---------|---------|
| Category | 问题分类 | 可选General Support或Billing Question |
| Server | 关联服务器 | 如无关请填写None |
| Subject | 问题标题 | 简明扼要概括问题 |
| Message | 详细描述 | 包含错误信息、复现步骤等 |

**小贴士**：上传相关截图或日志文件能显著提高解决效率。
